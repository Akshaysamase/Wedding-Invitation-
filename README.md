# 💍 Wedding Landing Page — Akshay & Rupali

A beautifully crafted wedding invitation website designed to celebrate  
**Akshay & Rupali’s special day** with elegance and simplicity.

![Wedding Thumbnail](image/A&R.png)

---

## ✨ Features

🎉 Elegant landing section with couple details  
📱 Fixed bottom navigation for easy access  
⏳ Live countdown timer to the wedding day  
📍 Event date, time & venue details  
🗓️ Add to Calendar (Google Calendar integration)  
🗺️ One-click Google Maps direction  
💬 WhatsApp contact buttons for quick communication  
🎨 Smooth animations with AOS  
📱 Fully responsive (mobile-friendly)

---

## 🛠️ Tech Stack

- ⚡ **HTML5**
- 🎨 **Bulma CSS** — https://bulma.io
- ✨ **AOS (Animate on Scroll)**
- 📜 **JavaScript / jQuery**
